# ActiveError

## Usage

```ruby
ActiveError.new(StandardError, "error message")
```

## Installation

Add this line to your application's Gemfile:

```ruby
gem "active_error"
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install active_error
